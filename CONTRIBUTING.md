# Contributing

感谢你参与 `openclaw-mem0`。

## 贡献方式

1. Fork 仓库并创建分支：`feat/xxx`、`fix/xxx`
2. 提交前执行自检：
   - 脚本可执行
   - README 命令可复现
   - 不含明文密钥
3. 提交 PR，说明：
   - 变更目的
   - 变更内容
   - 验证方式

## Commit 建议

- `feat:` 新功能
- `fix:` 修复
- `docs:` 文档
- `refactor:` 重构
- `chore:` 杂项

## 安全约束

- 不提交真实 API Key / Token
- 不提交生产环境 IP 与内部域名
- 日志中若含用户隐私，请先脱敏

## Issue 建议模板

请至少提供：

- 环境信息（OS、Node、Python）
- OpenClaw / Mem0 版本
- 复现步骤
- 关键日志（脱敏后）
